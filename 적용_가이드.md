# 적용 가이드

## 1. APK 만들기

이 프로젝트는 소스 형태입니다. 아래 둘 중 하나로 APK를 만듭니다.

### 가장 쉬운 방법: GitHub Actions

1. GitHub에서 **새 비공개 저장소**를 만듭니다.
2. 압축을 풀고 폴더 안의 파일을 전부 업로드합니다. `.github` 폴더도 포함합니다.
3. 저장소의 **Actions** 탭을 엽니다.
4. 왼쪽에서 **APK 빌드**를 선택하고 **Run workflow**를 누릅니다.
5. 빌드가 끝나면 실행 결과 아래 **Artifacts**에서 `black-walnut-notes-debug-apk`를 내려받습니다.
6. 압축 안의 `app-debug.apk`를 휴대폰에 옮겨 설치합니다.

> GitHub로 빌드한 파일은 테스트용 debug APK입니다. 장기 사용용 서명 APK는 Android Studio의 `Build > Generate Signed App Bundle or APK`에서 본인 키로 만드는 것이 가장 안전합니다.

### Android Studio

1. Windows용 Android Studio를 설치합니다.
2. **Open**으로 이 프로젝트 폴더를 엽니다.
3. 초기 구성과 Gradle 동기화가 끝날 때까지 기다립니다.
4. `Build > Build Bundle(s) / APK(s) > Build APK(s)`를 선택합니다.
5. 생성 위치: `app/build/outputs/apk/debug/app-debug.apk`

## 2. 메모 파일 준비 및 Windows 동기화

### Windows

1. `setup_windows.ps1`을 우클릭해 **PowerShell에서 실행**합니다.
2. 다음 파일이 만들어집니다.
   - `%OneDrive%\BlackWalnutNotes\home.md`
3. 이후 Windows에서는 메모장, VS Code 등으로 이 파일을 편집합니다.

### Android: FolderSync 무료판

1. FolderSync에서 OneDrive 계정을 추가합니다.
2. 휴대폰에 `/Documents/BlackWalnutNotes` 폴더를 만듭니다.
3. **폴더페어(Folderpair/동기화 쌍)**를 추가합니다.
   - 계정: OneDrive
   - 원격 폴더: `/BlackWalnutNotes`
   - 로컬 폴더: `/Documents/BlackWalnutNotes`
   - 동기화 방식: **양방향**
   - 충돌 처리: **최신 파일 우선**보다 `충돌 시 건너뛰기/알림` 권장
4. 한 번 수동 동기화를 실행해 `home.md`가 휴대폰에 내려오는지 확인합니다.

메뉴 번역은 FolderSync 버전에 따라 `폴더페어`, `Folderpairs`, `동기화 쌍`처럼 다르게 보일 수 있습니다.

## 3. Android 편집기와 위젯 연결

1. Markor를 설치합니다.
2. Markor에서 `/Documents/BlackWalnutNotes`를 작업 폴더로 지정하거나 `home.md`를 직접 엽니다.
3. Black Walnut Notes APK를 설치하고 한 번 실행합니다.
4. 스마트 런처 홈 화면의 빈 곳을 길게 누릅니다.
5. **위젯 > Black Walnut Notes > Black Walnut 메모**를 선택합니다.
6. 설정 화면에서 `/Documents/BlackWalnutNotes/home.md`를 선택합니다.
7. **화면 일치** 프리셋을 누르고 저장합니다.

기본값:

- 배경 `#000000`
- 제목 `#5A3021`
- 본문 `#3A2017`
- 강조 `#D1AE6F`

본문이 너무 어두우면 위젯의 톱니바퀴를 눌러 **가독성** 프리셋으로 바꿉니다.

## 사용법

- 위젯 본문 터치: 연결된 메모 편집 앱으로 열기
- ↻ 버튼: 파일을 즉시 다시 읽기
- 제목 또는 톱니바퀴: 색상·파일·글자 크기 변경
- 자동 갱신: Android가 허용하는 범위에서 약 30분 주기

Windows에서 수정한 직후에는 FolderSync가 휴대폰 파일을 갱신한 다음 위젯의 ↻를 누르면 됩니다.

## 색상 프리셋

위젯 설정 화면의 **색상 프리셋**에서 다음 조합을 바로 선택할 수 있습니다.

- 블랙 월넛
- 다크 브라운
- 앤티크 브론즈
- 앤티크 골드
- 딥 앰버
- 번트 코퍼
- 골드 가독성

프리셋을 고른 뒤에도 배경·제목·본문·강조 색상의 HEX 값을 직접 수정할 수 있으므로, 스마트 런처나 One UI 테마에 맞춰 미세 조정할 수 있습니다. 설정은 위젯마다 따로 저장됩니다.
